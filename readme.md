


# Problem

Users who work offshift occasionally lock themselves out of their Active
Directory account. Although the IT department maintains an on-call rotation,
IT does not have to be the bottleneck.

# Solution

An automated self-service account unlock using two forms of identity provides the
necessary security verification for account unlock requests. By utilizing basic
MFA principles, identity can be proven using various factors of a person, such as 
where they are, who they are, or what they know.
